
Blood Donation Management System

A brief description of what this project does and who it's forBlood Donation Management System (BDMS)

Project Overview
The Blood Donation Management System (BDMS) is a web-based application designed to manage and organize blood donation activities efficiently. It connects blood donors, recipients, and administrators in one platform to make blood donation faster, easier, and more accessible.

This system allows users to:
- Register as blood donors
- Submit blood donation requests
- View available donors and blood types
- Update and manage donor information
- Delete outdated or incorrect records

The system follows a CRUD (Create, Read, Update, Delete)** structure and provides a simple and user-friendly interface.

Objectives

- To simplify blood donation management
- To organize donor and recipient information
- To provide a fast and efficient system for blood requests
- To demonstrate CRUD operations in a real-world system

Technologies Used

- React.js (Frontend Framework)
- Vite (Build Tool)
- CSS (Styling)
- Stitch AI (UI/UX Design Tool)
- Netlify (Deployment)
